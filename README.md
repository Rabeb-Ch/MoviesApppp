# MoviesApp
IOS application that fetches movies from TMDB API and displays the list of trending movies, when we select one movie it shows more details about it.

# Technologies Used
* TMDB API
* Swift, UIKit
* Storyboards & XIB
* Combine

# Skills Demonstrated

* MVVM Pattern
* Closures
* Reactive programming
* API Integration
* JSON Decoding
* Error Handling
